# This is an app that gets the train arrival times for all MTA subway and LIRR stops. The user can save stations to their favorites (via MongoDB) for quicker access to the stations that they often use.

# APIs used - NYC Subway Times - https://api-endpoint.mta.info/Dataservice/mtagtfsfeeds/nyct%2Fgtfs, LIRR Times - https://api-endpoint.mta.info/Dataservice/mtagtfsfeeds/lirr%2Fgtfs-lirr
